# Library Management System

A simple **console-based Library Management System** developed in **Java**. This project allows librarians to manage books, patrons, borrowing, returning, and transaction records using Java Collections.

## Features

* Add new books
* Remove books
* View all books
* Search books by title or author
* Add patrons
* Remove patrons
* View all patrons
* Borrow books
* Return books
* Maintain transaction history
* Menu-driven console interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* HashMap
* ArrayList
* Scanner
* LocalDateTime

## How to Run

### Compile the project

```bash
javac *.java
```

### Run the application

```bash
java LibraryManagementSystem
```
